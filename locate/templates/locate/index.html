Refuge

This project aims to create a safe and inclusive environment for all individuals by providing a directory of restrooms that are verified to be safe and accessible for trans and non-binary individuals.

This website is easy to use and allows users to search for restrooms based on location and zip code, and users can also comment on existing restrooms.

I hope that this service will help to promote inclusion and accessibility in public spaces, and make it easier for everyone to find safe and comfortable restrooms when they need them.
